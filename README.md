# Work Order System

アルマイト工場の現場業務を想定した  
製造指示書（ラッキング工程）管理システムです。

## ■ 開発背景

現場では現在、紙ベースで製造指示書を管理しています。

- 事務所が製図をコピー
- 製造番号・仕様・本数を手書き
- 枠ごとに管理
- 完了時間や最終チェックも紙記録

この運用では、

- 転記ミス
- 集計の手間
- 過去履歴の検索の難しさ

といった課題があります。

これをWebシステム化し、
タブレット入力前提で現場負担を減らすことを目的としています。

---

## ■ 想定機能（設計段階）

- 枠単位での指示書作成
- 品番ごとの数量管理（例：5/8）
- ラッキング完了時刻入力
- 最終チェック記録
- 管理番号による検索

---

## ■ 技術構成

- Laravel 12
- PHP 8.5（Sail）
- MySQL 8.4
- Docker（Laravel Sail）
- Breeze（認証）

## ■ 開発方針

実務現場を前提とした設計を重視しています。

- 入力負担を増やさない
- 紙運用との差を最小化
- 現場優先のワークフロー設計
